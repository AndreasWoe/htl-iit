# LCD1602 an Raspberry Pi 2 I2C Bus

## Enable I2C on Raspberry

Start / Preferences / Raspberry Pi Configuration / Interfaces

## Test I2C / Get address of I2C device

```
i2cdetect -y 1
```

The `-y` option in the i2cdetect command is used to skip the interactive confirmation prompt. `1` is the I2C bus name.

## Build and Run library / demo program

```
git clone https://github.com/bitbank2/LCD1602
make
make -f make_demo
./demo
```

Achtung !!! - in der Library ist ein Fehler - der Befehl für "Clear" darf nicht 0x0E lauten, sondern muss auf 0x01 geändert werden!
